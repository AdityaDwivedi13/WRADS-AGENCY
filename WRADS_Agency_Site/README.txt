=====================================================
HOW TO RUN LOCALLY:
=====================================================

OPTION 1: Direct Browser Opening (Simple)
-----------------------------------------
1. Navigate to the WRADS_Agency_Site folder
2. Double-click on "index.html"
3. The website will open in your default browser

Note: The 3D globe requires an internet connection to load
the Three.js library and Earth texture from CDN.

OPTION 2: Local Server (Recommended for full features)
-------------------------------------------------------
Using Python:
  1. Open terminal/command prompt in this folder
  2. Run: python -m http.server 8000
  3. Open browser to: http://localhost:8000

Using Node.js:
  1. Install http-server: npm install -g http-server
  2. Run: http-server
  3. Open browser to: http://localhost:8080

Using VS Code:
  1. Install "Live Server" extension
  2. Right-click index.html -> "Open with Live Server"
